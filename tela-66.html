ESTRATÉGIAS PARA PROMOVER UMA GESTÃO PARTICIPATIVA – PARTE 2

Nesse contexto, são necessárias outras fontes de informação, não só as geradas pela agregação dos dados do e-SUS APS, mas por sistemas que captem a existência de características da coletividade, tanto variáveis emergentes de interações entre os sujeitos quanto redes, valores e formas de organização sociais, ambientais, que contextualizem as condições de vida no espaço geográfico. Isso pode ser feito de diversas maneiras, uma delas é por meio de dados secundários, como os das agências de governo, que possuem base de dados usados para gerenciar suas ações, censos e pesquisas organizados.
